# Sniper - Network Testing Tool

Sniper is an advanced tool for conducting various types of network attacks, such as HTTP GET, DNS Request, SYN Flood, or UDP Flood. The tool is designed for penetration testing and network security assessment purposes.


## Installation


Clone the repository:

    git clone https://github.com/jrdpteam/Sniper.git

Navigate to the project directory:

    cd Sniper

Install dependencies(may take a lot of time,depends on Your internet speed):

    sudo bash install_dependencies.sh

Run the setup script using this command:
    
    sudo python3 setup.py

Follow the instructions provided during installation to complete the setup.


## Usage


Once installed, Sniper can be invoked from the command line using this command:

    sudo Sniper

Features

   -  HTTP GET
   -  DNS Request
   -  ARP Request
   -  SYN Flood
   -  HTTP POST
   -  ICMP Echo Request
   -  UDP Flood
   -  Slowloris Attack
   -  DNS Poisoning Attack
   -  NTP Amplification Attack
